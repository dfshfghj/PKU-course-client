# PKU-Course-Client

PKU-Course-Client（北大教学网客户端）旨在美化并扩展北京大学教学网(https://course.pku.edu.cn)，支持数据下载至本地以离线查看。

## 功能

目前支持的主要页面：

+ 课程通知

+ 课程教参

+ 课堂实录

+ 课程作业

+ 个人成绩
